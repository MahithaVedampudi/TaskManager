⚡ QUICK START (3 COMMANDS)
═════════════════════════════════════════════════════════════════════════════

First, you need to have MySQL running. Choose one:

Option A - Using Docker (Easiest):
  docker run --name task_manager_db -e MYSQL_ROOT_PASSWORD=password \
    -e MYSQL_DATABASE=task_manager -p 3306:3306 -d mysql:8.0

Option B - Local MySQL (if already installed):
  mysql -u root -p
  > CREATE DATABASE task_manager;

Then run these 3 commands:

  1. pnpm install
  2. pnpm db:push && pnpm db:setup
  3. pnpm dev

Open: http://localhost:3000
Login with credentials above ✓


🚀 DEPLOY TO RAILWAY (LIVE URL)
═════════════════════════════════════════════════════════════════════════════

Want your app live? Follow these 4 steps:

1. Push code to GitHub:
   git add . && git commit -m "Task Manager" && git push

2. Go to: https://railway.app
   Sign up with GitHub

3. Create new project:
   Click "Create New Project" → "Deploy from GitHub repo" → Select your repo

4. Setup:
   Add MySQL service, set environment variables (JWT_SECRET, NODE_ENV, URL)
   
   Generate JWT_SECRET: openssl rand -hex 32

Done! Your app is live in 5 minutes 🎉

🆘 TROUBLESHOOTING
═════════════════════════════════════════════════════════════════════════════

Can't connect to database?
  → Verify MySQL is running: docker ps | grep task_manager
  → Check .env.local DATABASE_URL is correct
  → Run: pnpm db:push

Login not working?
  → Clear browser cookies (F12 → Application → Cookies)
  → Hard refresh (Ctrl+Shift+R)
  → Verify JWT_SECRET in .env.local

Tasks not showing?
  → Ensure logged in as admin
  → Run: pnpm db:setup to create sample data
  → Check browser console for errors

Railway deployment fails?
  → Check deployment logs in Railway dashboard
  → Verify .env.local is NOT committed
  → Ensure all environment variables are set

For more help:
  → See SETUP_GUIDE.md for detailed troubleshooting
  → See DEPLOYMENT.md for Railway-specific issues
