🚀 QUICK START (3 STEPS)
═════════════════════════════════════════════════════════════════════════════

STEP 1: Setup MySQL (Choose One)

  Docker (Recommended):
    docker run --name task_manager_db \
      -e MYSQL_ROOT_PASSWORD=password \
      -e MYSQL_DATABASE=task_manager \
      -p 3306:3306 -d mysql:8.0

  Local MySQL:
    mysql -u root -p
    > CREATE DATABASE task_manager;

STEP 2: Run These Commands

  pnpm install
  pnpm db:push && pnpm db:setup
  pnpm dev

STEP 3: Open Browser

  http://localhost:3000
  Login with: admin@taskmanager.com / admin123
